Tentang Aplikasi

Aplikasi Analisis Curah Hujan

Aplikasi berbasis web untuk mengevaluasi efisiensi dan efektivitas dataset curah hujan dengan model klasifikasi berbasis machine learning.

Aplikasi ini memungkinkan pengguna untuk mengunggah dataset meteorologi, melakukan analisis komprehensif, membangun model klasifikasi curah hujan, dan mendapatkan rekomendasi untuk meningkatkan kualitas dataset.

Fitur Utama

  • Analisis kepentingan fitur meteorologi
  • Perbandingan berbagai algoritma klasifikasi
  • Tuning model untuk performa optimal
  • Evaluasi efisiensi dan efektivitas dataset
  • Prediksi kategori curah hujan
  • Ekspor model untuk penggunaan eksternal

Algoritma yang Digunakan

  • Random Forest
  • XGBoost
  • Support Vector Machine (SVM)
  • Logistic Regression
  • Gradient Boosting
  • K-Nearest Neighbors (KNN)

Panduan Penggunaan

Unggah file CSV atau Excel yang berisi data meteorologi Anda. Dataset harus berisi kolom-kolom yang terkait dengan parameter meteorologi seperti arah angin, kecepatan angin, kelembapan, temperatur, dan curah hujan.

  • Format yang didukung: CSV, XLSX, XLS
  • Ukuran file maksimum: 16MB
  • Pastikan data Anda bersih dan tidak memiliki nilai yang hilang yang signifikan

Aplikasi akan melakukan pra-pemrosesan data yang meliputi normalisasi, penanganan nilai yang hilang, dan definisi kategori curah hujan. Anda dapat mengatur parameter pra-pemrosesan seperti:

  • Kolom target (curah hujan)
  • Batas kategori curah hujan
  • Label untuk setiap kategori

Aplikasi akan menganalisis kepentingan fitur-fitur dalam dataset untuk menentukan fitur mana yang paling berpengaruh terhadap prediksi curah hujan. Anda dapat memilih metode analisis kepentingan fitur:

  • Random Forest: Menggunakan algoritma Random Forest untuk mengevaluasi kepentingan fitur
  • ANOVA F-test: Menggunakan uji statistik untuk mengukur korelasi fitur dengan target
  • RFE: Secara bertahap menghilangkan fitur yang kurang penting

Aplikasi akan membangun dan membandingkan berbagai model klasifikasi untuk menemukan model terbaik untuk dataset Anda. Model-model yang dibandingkan meliputi:

  • Random Forest
  • XGBoost
  • Support Vector Machine (SVM)
  • Logistic Regression
  • Gradient Boosting
  • K-Nearest Neighbors (KNN)

Aplikasi akan mengevaluasi efisiensi dan efektivitas dataset Anda berdasarkan performa model. Evaluasi meliputi:

  • Pengaruh ukuran dataset terhadap performa model
  • Pengaruh jumlah fitur terhadap performa model
  • Identifikasi fitur optimal
  • Rekomendasi untuk meningkatkan kualitas dataset
